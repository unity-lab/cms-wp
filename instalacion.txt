Paso obligatorio premiliminar a cualquier accion:

    1. Ubicarse con la consola adentro de la carpeta del proyecto de git por ejemplo  c:/misproyectos/cms-wp

    2. INSTALAR Y EJECUTAR DOCKER DESKTOP (en windows) de lo contrario no funcionara la terminal al ejecutar comando docker

    3. Copiar ".env.dev" en esta misma carpeta y renombrar el archivo a ".env"

    4. Ejecutar:  docker compose up para que se levanten los contenedores (mysql y phpmyadmin)

    5. En el navegador colocar http://localhost:8090/ 
        para entrar al contenedor de PHPMYADMIN que se esta ejecutando y acceder al GUI de mysql
        aqui configure puertos 8090 en la variable de entorno para que no haya conflicto con otras versiones 

    6. En otra ventana o pesta√±a del navegador colocar http://localhost:8000 (TIENE DEMORA como 3 o 5 minutos EN CARGAR)
            http://localhost:8000/wp_admin para acceder al menu de usuarios de wordpress
    
    7. IMPORTANTE: 
            cada vez que retomamos el proyecto: 
            docker compose down 
            y luego 
            docker compose up 
            en un terminal (SOLO si hubo cambios de configuraciones)

    8. en la terminal donde se ejecuta docker apretar control + c para detener la ejecucion de docker (y sus contenedores)

    9. cuando modifiques el proyecto debera exportar la base de datos y sobreescribir la que existe en el directorio backup_DB
